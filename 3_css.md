## cssを書いてみる
* key: value の形式で書く
* htmlタグに書く
  ```html
  <p style="color: red;"></p>
  ```
* htmlファイルのheaderに書く
  ```html
  <style type="text/css">
    p { color: red; }
  </style>
  ```
* 別ファイルに書く
  ```
  p { color: red; }
  ```
